#REST Scaffold
Scaffolding to quickly start up a RESTful node server.

## Includes
1. Sequelize : ORM
2. PostgreSQL : Database
3. Express : Server
4. Lodash : Scripting utility
5. Mocha, Chai : Testing
6. Passport : Authentication

##Running Development
Start the dev server
```$
    npm run dev 
```

Start dev server in Debug mode
```$
    npm run debug
```

##Running Tests
```$
    npm run test
```
